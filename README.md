# repeat

repeat a command and be alerted when the output changes

## 🚀 Installation

with [basher](https://github.com/basherpm/basher)

	$ basher install pforret/repeat

or with `git`

	$ git clone https://github.com/pforret/repeat.git
	$ cd repeat

## 🔥 Usage

	> repeat.sh [action]
